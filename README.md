# 🚀 سیستم وبلاگ پیشرفته جنگو (معماری Hybrid: CBV + DRF)

یک پروژه آموزشی و صنعتی همزمان، توسعه داده شده با **Django** و **Django REST Framework (DRF)**. این پروژه با هدف تسلط عمیق بر کلاس‌بیس ویوهای جنگو (CBVs)، تفکیک سیستم احراز هویت، و پیاده‌سازی یک معماری چند کانتینری مجهز به **PostgreSQL** و **Redis** طراحی و پیاده‌سازی شده است.

---

## ✨ ویژگی‌های کلیدی و امکانات پروژه

### ۱. بخش کاربران و احراز هویت (Accounts App)
- **Custom User Model:** حذف کامل سیستم پیش‌فرض `username` و جایگزینی آن با لایه احراز هویت ایمیل‌محور (`email` به عنوان USERNAME_FIELD).
- **سیستم پروفایل خودکار:** تفکیک اطلاعات حیاتی کاربر از اطلاعات جانبی با ایجاد مدل `Profile` مجزا.
- **اتصال با Signal:** ساخت و مدیریت خودکار پروفایل کاربران به محض ثبت‌نام در سایت از طریق سیگنال نیتیو `post_save`.

### ۲. بخش وبلاگ و مدیریت محتوا (Blog App)
- **مدل‌سازی سئومحور:** استفاده از فیلد `slug` به جای آیدی عددی (`pk`) برای دسترسی به صفحات جزئیات مقالات جهت بهبود رتبه‌بندی موتورهای جستجو.
- **مدیریت وضعیت با TextChoices:** پیاده‌سازی استاندارد نوظهور `models.TextChoices` برای فیلد وضعیت مقالات (حالت‌های پیش‌نویس `DR` و منتشر شده `PB`).
- **دسته بندی داینامیک:** ارتباط چند به یک مقالات با مدل `Category` همراه با سیستم `prepopulated_fields` در پنل ادمین جهت تولید خودکار اسلاگ‌ها.

### ۳. معماری فنی و زیرساخت (DevOps & Databases)
- **Hybrid Backend:** توسعه همزمان لایه مونو‌لیت (قالب‌های HTML جنگو با لایه CBV) و لایه API قدرتمند با **DRF** جهت سرویس‌دهی به اپلیکیشن‌های موبایل یا فرانت‌اندهای مدرن.
- **Multi-Container Docker:** ایزوله‌سازی و مدیریت کامل تمام سرویس‌ها (وب، دیتابیس و کش) در یک ساختار یکپارچه با `docker-compose.yml`.
- **PostgreSQL 15:** بهره‌گیری از پایگاه داده لایه صنعتی و پایدار پورت‌اس‌کیوال نسخه Alpine جهت بهینه‌سازی ذخیره‌سازی داده‌ها.
- **Redis 7:** آماده‌سازی و کانفیگ کانتینر ردیس به عنوان In-Memory Database جهت راه‌اندازی سیستم کَشینگ، مدیریت Sessionها و کارهای ناهمگام (Celery).
- **مدیریت فایل‌های رسانه‌ای:** پیکربندی داینامیک مسیرهای `STATIC` و `MEDIA` برای آپلود و نمایش زنده تصاویر مقالات در حالت توسعه (`DEBUG=True`).
- **بین‌المللی‌سازی:** استفاده از سیستم ترجمه بومی جنگو (`gettext_lazy`) برای تمامی فیلدها و بخش‌های پروژه.

## 🛠 تکنولوژی‌ها و ابزارهای به‌کار رفته

- **Frameworks:** Django 5.x | Django REST Framework (DRF)
- **Databases:** PostgreSQL 15 | Redis 7
- **DevOps:** Docker | Docker Compose
- **Python Libraries:** `psycopg2-binary` (اتصال به پورت‌اس‌کیوال)

├── docker-compose.yml     # ارکستراسیون سرویس‌های Web, DB, Redis
└── requirements.txt       # لیست نیازمندی‌ها و پکیج‌های پایتونی
